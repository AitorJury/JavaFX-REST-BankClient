Window description

This window allows you to make payments and deposits from your previously selected bank account. The user can use the controls available in this window.

Controls

When you access the transactions window, you'll see a list of transactions. In the upper left corner, you'll see your account ID, and below, you'll see a series of buttons:

This is the transaction type selector, followed by the text field for entering the amount and the "Create" button, which remains disabled until the amount is entered. It's important to note that the "Create" button will not work if the amount is negative or contains non-numeric characters.

The "Delete" button will only remove the last transaction from the account and will be deactivated after use. If you create a new transaction, it will be reactivated.

The "back to accounts" button will take you to the previous page where you can view your accounts.
