Basic Description

This window allows you to carry out maintenance operations on basic user data. The user will be able to use the controls available in the creation, modify and delete users window. The application. Likewise, you can generate a report that contains a list of users with their corresponding data.

Controls

When you enter the user administration window, you have several options to perform with existing users: create, edit, and delete. Upon entering, you will see a green button to add a user.

An empty user will be created with a random password and email address. It is recommended to assign the data before exiting the application to avoid saving an empty user.

The user can be edited by clicking on the user row. The user data will be displayed in the fields.

The user can be deleted by clicking the red button. It's labeled "Delete Client." A dialog box will open asking: "Are you sure you want to delete this user?" You can then confirm or cancel the deletion.

If you try to delete a user with an associated account, you will get an error message because deleting a user with an associated account is not allowed.

If you don't have any rows selected, the delete button will be disabled.
